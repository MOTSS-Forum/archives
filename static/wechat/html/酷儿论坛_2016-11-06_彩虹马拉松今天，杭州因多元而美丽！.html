| 多元才美丽 |


七点半，马拉松正式开始时。我们大部队举着标志性的彩虹大旗，在人群中显得格外耀眼。不少镜头都对着我们的旗帜和标语不停按快门记录着。运动员与志愿者们一齐高声呼喊着我们事先准备好的口号：


“家家都有性少数，社会需要包容度。

不男不女不一样，又男又女又怎样？

撑同志，反歧视。

爱情不分性别，同性一样和谐。

家家都有同性恋，只是你们看不见。

不止能顶半边天，妇女能顶整片天。

打非亲，骂非爱，告别暴力和伤害。

杭州因多元而美丽。”

 

我们的标语和口号，涉及了性少数平权、女权主义、反暴力等等。


我们的世界，我们的声音，在红旗飘扬的地方都有爱，都有平等。

| 家家都有性少数 |



除了彩虹旗以外，我们还选取了几面代表少数中的少数的旗帜，比如无性恋、跨性别等。这是因为向阳花开认为，不仅仅只是同性恋，任何群体的权益都需要被尊重。笔者曾经看过一部英国电影《Pride》，里面讲的是英国平权史上，同性恋与工人群体从偏见到包容，从误解到联盟的故事。如何去联合所有的少数群体，一起争取平等权益，或许是国内平权运动的下一个突破口。


不少朋友表示，通过参与我们组织的活动，自我认同得到了很大的进步，从一开始的畏惧与“深柜”，到走出柜子去为自己的权益努力。有人说在这样一个群体中，得到了真正意义上的归属感；有人说，看到亲友会的妈妈们也来支持我们的活动，内心非常感动，因为获得家人的认同与支持正是我们一直以来渴望的；有人说，通过彩虹马拉松，结识了志同道合的好伙伴，甚至在这里找到了属于自己的爱情；也有人说，来年还想继续参加彩虹马拉松，那种和大家在一起在公众场合下做倡导的感觉太棒了。

值得注意的是，在路上我们遇见不少异性恋叔叔阿姨们对我们竖大拇指，为我们加油，给我们支持。我们一直倡导异性恋群体加入彩虹公益，很多人不解，因为大家都把彩虹公益定义为一场性少数社群内部的狂欢。其实不然，与女权主义一样，多元也涉及每一个人的利益。在某个节点，不管什么性别和性取向，我们任何人都有可能成为少数而面临被孤立的境况。



▲不只是LGBT+的狂欢


除了性少数群体会遭遇污名化，异性恋女性也在当下以父权制为主体的社会氛围中深受其害。 “二孩政策”的出台，舆论对于不结婚不生育女性的恶意，对于女博士、女政客、女司机的污名化在新闻中可见一斑。即便是在这种二元对立传统文化中被认为是“既得利益”阶层的男性群体也一样是深受其害的，社会认为男性必须要有担当，有房有车有钱有地位，作为一个保护者出现在公众眼中。假如一个男性想要回归家庭做一个“奶爸”，没有那么强烈的事业心，他也会遭受舆论的谴责，会被定义为一个懦弱、缺乏责任心和事业心的人。这就变相导致了父亲角色在很多家庭中的缺位。


笔者曾经看过一个外国纪录片，里面有人采访了支持同志公益的年轻直人妈妈，追问其支持的原因。她回答：“如果有天我的孩子长大变成了gay，我不希望他的周围环境依然是充满歧视的。”由此可见，倡导多元、支持性少数权益并非只是LGBT+群体内部的狂欢，更是公众需要去倡导与支持的事。

 

▲在彩虹旗下的孩子&围观的路人


如果有一天，多元理念被社会大众所采纳，就不会有千千万万的同妻同夫无法获得法律保障而饱受无爱婚姻的摧残，就不会有相恋几十年的同性情侣在一方病重之时无权在手术书上签字，就不会有异性恋男性一个人背负着养家糊口的重担，就不会有大龄单身的女性背负着舆论压力在最好的年华为自己的年龄焦虑。

 

▲今天清晨第一缕阳光后扬起的彩虹旗


全美婚姻合法化之后，大家都在为这人类历史上的壮举而感叹着、艳羡着。然而我们却忽视了，在全美婚姻合法化的背后，是千千万万个体与LGBT+公益组织的共同努力的结果。每一段历史的前进都是由一点一滴的微小努力汇合而成的。正是那些在人群中呐喊狂奔的性少数群体，为后代争取到了平等的婚姻权、就业权、抚养权。

 

▲微小却又无比强大的每个人


今天，我们向阳花开在杭州马拉松队伍中高举彩虹旗，为性少数的权益拼搏呐喊。作为个体我们能做的也许是微乎其微的，但那种身披彩虹旗尽情呼喊的快意，那种发挥个人主观能动性去争取幸福权利的精神，让我们每个人的英雄主义都发挥到了极致。最终，所有看似微小的努力都会汇成一股巨大的力量，去推动整个社会的变革。终有一日，我们会牵着心爱之人的手，在大街上尽情地拥吻，在这片生我养我的国土上，拥有平等的权益。

 


▲今年参与人数又创新高，镜头都快装不下了

If not me, who? 如果不是我，那又该是谁？

If not now, when? 如果不是现在，那又该是何时？
